# Gustavo Font Files

Place Gustavo font files here. The dashboard uses:

- `Gustavo-Regular.woff2` / `Gustavo-Regular.woff`  (weight 400)
- `Gustavo-Medium.woff2`  / `Gustavo-Medium.woff`   (weight 500)
- `Gustavo-Bold.woff2`    / `Gustavo-Bold.woff`      (weight 700)

Gustavo is a commercial font by Latinotype.
Available at: https://www.myfonts.com/collections/gustavo-font-latinotype

Without the font files, the dashboard falls back to Helvetica Neue / Arial.
